# For Someone Special

A romantic static website that can be hosted directly on GitHub Pages.

## Customize

- Edit the words in `index.html`.
- Replace `Beautiful` with her name, or let the page ask for her name.
- Replace `assets/romantic-terrace.png` if you want a different background.

## Publish on GitHub Pages

1. Create a new GitHub repository.
2. Upload these files to the repository root.
3. Open the repository on GitHub.
4. Go to **Settings > Pages**.
5. Under **Build and deployment**, choose **Deploy from a branch**.
6. Select the `main` branch and `/root`, then save.

Your site will be available at:

```text
https://YOUR-GITHUB-USERNAME.github.io/YOUR-REPOSITORY-NAME/
```
